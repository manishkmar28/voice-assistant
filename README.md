# 🎙️ EchoMind AI

EchoMind AI is a cross-language Voice Assistant built using Java, Spring Boot, and Python. It recognizes voice commands and performs desktop automation tasks such as opening applications, launching websites, and executing system commands.

## ✨ Features

- 🎤 Voice Recognition
- ☕ Java Spring Boot Backend
- 🐍 Python Speech Processing
- 🌐 Open Websites
- 📝 Open Notepad
- 🧮 Open Calculator
- 💻 Open Command Prompt
- ⚡ Desktop Automation
- 🔄 Java & Python Integration

## 🛠️ Tech Stack

- Java 21
- Spring Boot
- Python
- SpeechRecognition
- PyAudio
- Maven
- HTML
- CSS

## 🚀 Getting Started

1. Clone this repository.
2. Open the project in Eclipse.
3. Install Python dependencies:
   ```bash
   pip install SpeechRecognition PyAudio requests
   ```
4. Run the Spring Boot application.
5. Run `voice_client.py`.
6. Give voice commands to control your system.
